Enable recurring achievements with an optional daily limit

== Description ==
Recurring Achievements Ltd is a plugin to enable your [Achievements][] plugin to
allow unlocking achievements more than once, with the optional ability to have a
daily limit.

This plugin requires the excellent [Achievements][] plugin to work.

[Achievements]: http://achievementsapp.com/

== Installation ==
Download and install the plugin via the built-in plugins installer. No other
installation is required.
